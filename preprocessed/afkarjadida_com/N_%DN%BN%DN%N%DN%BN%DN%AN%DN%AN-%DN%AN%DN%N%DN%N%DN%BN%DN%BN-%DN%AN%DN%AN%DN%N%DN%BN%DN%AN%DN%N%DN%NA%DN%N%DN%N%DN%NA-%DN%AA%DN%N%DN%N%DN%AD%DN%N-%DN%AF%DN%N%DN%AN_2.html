    شوربة الفطر بالرافيولي تمنحك دفء

    مكوّنات الشوربة والرافيولي

    أعلنت الشيف ثريا الألفي عبر صفحتها الرسمية على فيسبوك عن طبق يجمع بين نعومة شوربة الفطر ورافيولي محشوة. وتتضمن المكوّنات 2 كوب فطر مقطع و1 بصلة صغيرة مفرومة و2 فص ثوم مهروس، إضافة إلى 2 ملعقة كبيرة زبدة و1 ملعقة كبيرة زيت زيتون. كما يحتاج الطبق إلى 3 أكواب مرق دجاج أو خضار وكوب كريمة طهي، مع ملح وفلفل أسود حسب الرغبة ورشة زعتر أو أوريغانو، إضافة إلى 250 جم من الرافيولي الجاهز (جبنة أو دجاج حسب الرغبة) وبقدونس للتقديم.

    يبدأ التحضير بتسخين الزبدة والزيت في قدر ثم إضافة البصل وتحميره حتى يذبل. يلي ذلك إضافة الفطر وتقليبه حتى يتحمر وتظهر نكهته. بعدها يُضاف الثوم ويُقلّب لمدة دقيقة، ثم تُسكب المرقة وتغلي الشوربة لمدة خمس دقائق. في الخطوة الأخيرة تُضاف الرافيولي الجاهز وتُترك للشوربة لتطهي 6–8 دقائق حتى النضج، ثم تصب الكريمة وتُتبّل بالملح والفلفل ورشة من الزعتر وتُقدم ساخنة مع رشة من البقدونس المفروم.
